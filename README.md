# OAuth Authentication with Next.js

This project is a practical implementation of OAuth authentication using Next.js. It demonstrates how users can securely sign in through third-party identity providers, manage authenticated sessions, and access protected resources.

## Features

* OAuth-based authentication
* Secure session management
* Protected routes and pages
* User profile retrieval
* Modern Next.js architecture

## Tech Stack

* Next.js
* OAuth 2.0
* TypeScript
* React

This project was created to gain hands-on experience with authentication, authorization flows, and secure web application development.
